# CivicPulse

CivicPulse is a web-based platform that helps citizens report, track, and monitor local civic issues. The system provides a simple and interactive interface for reporting problems and tracking their resolution status.

## Features

- Report civic issues
- Track issue status
- Interactive user interface
- Real-time issue statistics
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Leaflet.js (Map Integration)

├── style.css
├── app.js
